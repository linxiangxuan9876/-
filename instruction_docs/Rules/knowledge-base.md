# 知识库规则

## Q&A 分类体系

### 主分类（示例）
| 分类 | 说明 |
|------|------|
| 保修政策 | 保修期限、条件、流程 |
| 保养维护 | 定期保养项目、周期 |
| 故障诊断 | 常见故障及解决方案 |
| 配件更换 | 配件规格、更换周期 |
| 服务预约 | 预约流程、注意事项 |
| 门店活动 | 促销活动、服务优惠 |

### 子分类
- 按主分类细分
- 具体问题类型
- AI 自动生成 + 人工确认

## Q&A 状态流转

```
                       ┌──────────────┐
                       │ pending_     │
                       │ review      │
                       └──────────────┘
                              │
           ┌──────────────────┼──────────────────┐
           ↓                  ↓                  ↓
    ┌──────────┐       ┌──────────┐       ┌──────────┐
    │published │       │rejected  │       │(duplicate)│
    └──────────┘       └──────────┘       └──────────┘
           │                                       ↓
           │                              ┌──────────────┐
           │                              │ merge prompt │
           └─────────────────────────────→└──────────────┘
                       恢复后重新审核

    ┌──────────┐ 软删除 ┌──────────┐ 恢复 ┌──────────┐
    │ (deleted)│ ←───── │published │ ───→ │published  │
    │          │        └──────────┘      └──────────┘
    └──────────┘
```

## AI 自动分类

### 分类 Prompt
```python
CLASSIFY_PROMPT = """
请为以下 Q&A 内容分类：

问题：{question}
答案：{answer}

分类要求：
1. 主分类：从保修政策、保养维护、故障诊断、配件更换、服务预约、门店活动中选择
2. 子分类：根据内容生成一个简洁的子类别

返回 JSON 格式：
{{"main_category": "主分类", "sub_category": "子分类"}}
"""
```

### 分类准确性
- 基于 GPT-3.5-turbo
- Temperature = 0.3（较低随机性）
- 准确率约 90%

## 批量导入规则

### Excel 格式要求
| 列名 | 必填 | 说明 |
|------|------|------|
| 问题 | 是 | Q&A 的问题部分 |
| 答案 | 是 | Q&A 的答案部分 |

### 编码支持
- UTF-8
- UTF-8-sig
- GBK
- GB2312
- Latin1

### 跳过规则
- 包含"示例"、"测试"等关键词的行自动跳过
- 空白问题或答案跳过

## 审核规则

### 审核操作
| 操作 | 结果 | 说明 |
|------|------|------|
| 通过 | `published` | 直接发布 |
| 拒绝 | `rejected` | 拒绝发布 |
| 删除 | `is_deleted=true` | 移入回收站 |

### 批量审核
- 支持多选操作
- 支持批量通过/拒绝
- 操作结果可追溯
